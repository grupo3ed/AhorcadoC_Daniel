1º Creamos un fork a nuestro repositorio Github del repositorio Github en el quequeremos contribuir. De esta forma se eliminarán problemas de compatibilidad futuros con la rama master de ese repositorio.

2º Para descargar el repositorio Github en local hace falta introducir en la consola de Git el comando "git clone" y añadir la url de ese repositorio.

3º Ejecutamos "git add Examen-IDM1/" para ponerlo en situación de "staged" y que Git lo reconozca. 

4º Creamos nuestro propio directorio dentro de Examen-IDM1 con "mkdir daniel_camara" y dentro del mismo otro directorio para albergar las capturas de pantalla. 

5º Ejecutamos "touch ed.txt" para crear el fichero sobre lo que más gusta y disgusta dela asignatura Entornos de Desarrollo y lo modificamos con el editor de textos, en este caso VIM.

6º Ejecutamos "git commit -am'Mensaje del commit'" para que Git reconozca todos los cambios que hemos realizado.
